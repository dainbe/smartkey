## 抵抗の計算
```
Red : ( 5 – 2 ) / 0.02 = 150Ω   300
Green : ( 5 – 3.6 ) / ( ( 2000 / 2500 ) * 0.02 ) = 240Ω 500
Blue : ( 5 – 3.6 ) / ( ( 2000 / 7000 ) * 0.02 ) = 91Ω 200
```
